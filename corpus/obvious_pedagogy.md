# obvious_pedagogy.md

High-probability "obvious" responses for pedagogy, teaching, and learning
briefs. Used to anchor originality scoring in pedagogy runs.

---

- break it into small manageable chunks
- use spaced repetition
- start with fundamentals and build up
- teach by examples rather than rules
- use active recall not passive review
- flip the classroom so practice happens in class
- use projects instead of lectures
- peer teaching solidifies understanding
- gamify with badges and progress bars
- personalise the learning path to each student
- use video lessons for asynchronous access
- mix visual auditory and kinesthetic modes
- scaffold difficulty so each step is reachable
- start with a hook that makes the student curious
- connect the material to the student's real life
- use formative assessment throughout
- celebrate small wins
- feedback should be immediate and specific
- learning happens in community not in isolation
- set clear learning objectives at the start
- use worked examples before problems
- fade scaffolding as mastery grows
- teach metacognition explicitly
- use spiral curriculum revisiting topics deeper each time
- assess with authentic tasks not tests
